# Prayer Tools App - Features

**Last Updated:** October 2025

## Overview

Prayer Tools is a platform for managing and distributing daily prayer content to subscribers. This document describes all user-facing features available in the application.

---

## For Administrators

### Campaign Management

**Create and Manage Prayer Campaigns**
- Create unlimited prayer campaigns with custom names and descriptions
- Each campaign gets its own unique web address (e.g., prayertools.com/my-campaign)
- Toggle campaigns between active and inactive
- Set a default language for each campaign
- Edit or delete campaigns at any time

**Supported Languages**
- English
- Spanish (Español)
- French (Français)
- Portuguese (Português)
- German (Deutsch)
- Italian (Italiano)
- Chinese (中文)
- Arabic (العربية)
- Russian (Русский)
- Hindi (हिन्दी)

### Content Creation

**Rich Prayer Content Editor**
- Write daily prayer content using an intuitive block-based editor
- Schedule content for specific dates
- Create the same content in multiple languages
- View all content organized by date

**Content Blocks Available**
- Headers and paragraphs for standard text
- Scripture quotes with references
- Prayer prompts highlighted as call-outs
- Bullet and numbered lists
- Images with captions
- Videos from YouTube, Vimeo, and social media
- Tables for organizing information
- Prayer checklists
- Section dividers

**Multi-Language Content**
- Create content in any of the 10 supported languages
- Each date can have multiple language versions
- Easy translation workflow
- Language flags show which versions exist

### Subscriber Management

**View and Manage Subscribers**
- See all people who signed up for a campaign
- Search subscribers by name, email, or phone number
- Filter by delivery method (email, WhatsApp, or app)
- View detailed information for each subscriber

**Edit Subscriber Information**
- Update subscriber names and contact details
- Adjust delivery preferences and reminder times
- Change frequency settings (daily or weekly)
- Mark subscribers as unsubscribed
- Delete subscriber records

**Subscriber Information Available**
- Name and contact information
- Chosen delivery method (email, WhatsApp, or mobile app)
- Reminder frequency (daily, weekly, or custom)
- Preferred reminder time
- Selected days (for weekly reminders)
- Subscription date
- Unique tracking ID

### Admin Access

**Secure Login**
- Log in with email and password
- Secure session management
- Easy logout

---

## For Public Users

### Campaign Landing Pages

**Discover Prayer Campaigns**
- Visit a campaign at its unique web address
- Read campaign descriptions and information
- Access today's prayer content directly
- Sign up to receive daily prayer reminders

### Subscribe to Campaigns

**Flexible Signup Options**
- Enter your name and contact information
- Choose how you want to receive prayers:
  - Email delivery
  - WhatsApp messages
  - Mobile app notifications (coming soon)
- Set your preferred reminder time
- Choose delivery frequency:
  - Daily prayers every day
  - Weekly prayers on selected days
  - Custom schedules

**International Support**
- Phone number input supports all countries
- Select your country code for WhatsApp delivery

### Daily Prayer Fuel

**Access Today's Prayer Content**
- Automatically loads today's prayer content
- Beautiful, readable content formatting
- Scripture passages, reflections, and prayer prompts
- Images and videos embedded in content
- Switch between available languages

**Track Your Prayers**
- Click "I Prayed" button after completing prayers
- Records that you prayed today
- Simple way to build prayer habits

**Browse Past Prayers**
- View previous prayer content
- Click any past date to read that day's prayer
- Full archive of all prayer content
- Filter by your preferred language

### Visual Experience

**Multi-Language Interface**
- Switch the entire interface to your preferred language
- All buttons, labels, and messages translated
- Date and time formatting matches your language
- Right-to-left support for Arabic
